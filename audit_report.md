# Audit Report - 避難收容處所資料修正

## Issues Found + Corrections Made

### 1. 資料排序問題
**Issue:** 相同縣市的避難收容所沒有排列在一起
**Correction:** 
- 實作排序邏輯：完全相同的縣市名稱排在一起，之後按前三字相同再排序
- 重新編號序號以確保連續性

### 2. 座標邊界問題
**Issue:** 座標超出台灣本島邊界或落於海上
**Correction:**
- 設定台灣本島邊界：經度 120.000°-122.000°E，緯度 21.900°-25.300°N
- 精度調整為小數點後第三位（公尺等級）
- 金門縣、澎湖縣資料跳過邊界檢查
- 實作海上區域檢查邏輯

### 3. 字元編碼問題
**Issue:** CSV檔案可能存在字元編碼問題
**Correction:**
- 支援多種編碼：UTF-8、UTF-8-SIG、Big5、GBK、CP950
- 修正常見編碼問題字元（如 ï¼š → ：）
- 對所有字串欄位進行編碼清理

### 4. 管理人電話欄位問題
**Issue:** 電話號碼包含中文字元或科學記號格式
**Correction:**
- 移除電話號碼中的中文字符
- 處理科學記號格式轉換
- 格式化台灣電話號碼（0912-345678、02-12345678）

### 5. 座標精度問題
**Issue:** 座標精度要求達到公尺等級
**Correction:**
- 所有座標值調整為小數點後第三位
- 確保修正後的座標符合精度要求

## 修正統計
- **原始記錄數:** 待執行後統計
- **排序處理:** 按縣市及鄉鎮市區重新排序
